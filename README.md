# Get-i9

## Description

Get and save i9i9.to images.

## Usage

    Usage: get-i9 <URL>

    URL: URL of a blog entry of i9i9.to.
         e.g. https://i9i9.to/c/276449

## Build

This is written in Go Language. So, just build it:

    # cd get-i9
    # go build
 
 Or, use `go get`:

    # go get github.com/shinraminagi/get-i9
